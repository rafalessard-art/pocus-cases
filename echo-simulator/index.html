How this works

The motion is real. The way the heart walls move in each loop is taken from real cardiac MRI scans (the public ACDC dataset) and replayed on a clean heart model — so you're watching motion that genuinely happened in patients' hearts, not an animation drawn by hand.

The numbers are measured, not typed in. The ejection fraction (EF) and the other readings — the right-vs-left ventricle size ratio, TAPSE, and the septal index — are all calculated from the moving outline you see on screen. Pick a different case and the shape changes, so the numbers change with it.

The cases. Each preset is a real pattern: a normal heart; three heart attacks grouped by blocked artery (LAD, RCA, LCx); right-heart strain; and two cardiomyopathies — one dilated and weak, one thick-walled — both taken from individual real patients.

The three looks. Teaching is a clean, labeled diagram. Echo mimics the grainy grayscale of a real ultrasound (and is checked against real ultrasound clips so it reads like a genuine scan). Photoreal (beta) is an experimental AI version that adds lifelike texture — it can make a disease look milder than it really is, so use Teaching or Echo when you're actually sizing up a case. The numbers always come from the validated model.

A teaching tool — not for real diagnosis. The disease cases still need formal cardiologist review.
